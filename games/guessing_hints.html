This is the code for a guessing game. The program chooses a number between 1 and 1000.

The user then guesses a number, and the program tells the user if they picked right.

The program tells the user if they guessed too high or too low.

There is a way to play this such that you always guess right using 10 guesses at most.


import random

is_playing = True
while is_playing == True:
    guesses = 0
    start = 1
    end = 1000
    print(f"Guess what number I am thinking of between {start}-{end}")
    chosen = random.randint(start, end)
    guess = 0
    while guess != chosen:
        guesses += 1
        guess = int(input("Number: "))
        if guess < chosen:
            print(f"The number is higher than {guess}")
        elif guess > chosen:
            print(f"The number is lower than {guess}")
        else:
            print(f"Yes! I was thinking of the number {chosen}")

    print(f"It took you {guesses} to guess the correct number. Do you want to play again?")
    play_again = input("(y/n): ")
    if play_again == 'n' or play_again == 'no' or play_again == 'No':
        is_playing = False
print("Thanks for playing!")
