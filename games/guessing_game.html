This is the code for a guessing game. The program chooses a number between 0 and 10.

The user then guesses a number, and the program tells the user if they picked right.

import random

start_num = 1
end_num = 10
# we create a variable named chosen_num
chosen_num = 0
# we set chosen_num to a random number between 1 and 10
chosen_num = random.randint(start_num, end_num)
print("I picked a number between " + str(start_num) + " and " + str(end_num) + ".")
# create a guess variable to store the user's guess
guess = 0
# have the user enter in a number (which is returned as a string)
guess = input("Guess which number I picked: ")
# turn guess into an integer using the int() function.
guess = int(guess)
# check if the user's guess equals the chosen number
if guess == chosen_num:
    print("Wow you guessed right!")
else:
    print("Sorry, that's not the number I picked.")
